Final state machine for controling a fishing rod via a stepper motor.

## How to connect:
Connect the switch to one of the analog pins on the motor shield.
Connect the arduino and the motor shield to their respective voltage sources.

## How to run:
Specify the number of rotation you want the motor to do each time in the .ino file.
Specify the pin that the switch is connected to in the .ino file.
Upload the code to the arduino.
The motor speed can be adjusted while running.
